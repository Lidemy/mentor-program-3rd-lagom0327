## hw2：超基礎 Todo list


請你完成一個很簡單的 Todo List，需要有以下功能：

1. 身為使用者，我可以新增 todo item => 按 `Enter` 或按 + 按紐，沒有內容時無效，從上方新增
2. 身為使用者，我可以刪除 todo item => 按右方 X icon
3. 身為使用者，我可以標記某個 todo item 已完成  => 畫橫線害改變顏色，只要游標不要在 X 上，整個 todo item 都能標記；已勾選的 item 會移到最下方
